# Docker Hub Notes

## What is Docker Hub?
GitHub but for Docker images - store and share images with the world!

## Commands
- `docker login` - login to Docker Hub
- `docker build -t username/imagename .` - build with your username
- `docker push username/imagename` - upload to Docker Hub
- `docker pull username/imagename` - anyone can download your image
