# AWS S3 Notes

## What is S3?
S3 (Simple Storage Service) is object storage used to store files in the cloud.

## Key Concepts
- Bucket: Container to store files
- Object: File stored inside bucket
- Region: Location where bucket is created

## Commands Used

List buckets:
aws s3 ls

Create bucket:
aws s3 mb s3://rahul-devops-demo-12345

Upload file:
aws s3 cp file.txt s3://rahul-devops-demo-12345

Download file:
aws s3 cp s3://rahul-devops-demo-12345/file.txt .

Upload folder:
aws s3 cp app/ s3://rahul-devops-demo-12345 --recursive

## Learnings
- S3 is used for storage, backups, and CI/CD artifacts
- CLI allows automation of uploads/downloads
- Bucket names must be globally unique

## Extra Notes

Useful flags:
```
--recursive → copy entire folders
aws s3 rm → remove files from bucket
aws s3 sync → sync local folder with S3 bucket
```
